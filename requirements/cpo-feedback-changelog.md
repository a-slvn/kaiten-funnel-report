# Changelog: Интеграция фидбека CPO в спецификации воронки продаж

## Назначение документа

Трассировочный документ, связывающий каждый пункт из анализа ревью CPO (`/Users/slvn/ai/kaiten/feedback/Funnel_CRM_analysis.md`) с конкретными изменениями в спецификациях проекта. Является единым источником правды о том, **что** меняется, **почему** и **где**.

**Дата ревью CPO:** февраль 2026
**Участники ревью:** Slava (CPO), Lena (продакт/разработка), Andrey (дизайнер/продакт), Din (продакт-менеджер)

## Сводка изменений

| Параметр | Было | Стало |
|----------|------|-------|
| Единица воронки | Одна доска (`board_id`) | Всё пространство (`space_id`, несколько досок) |
| Первый заход | Экран настройки (стоп-экран), график не отображается | Best Guess -- график строится сразу, настройка опциональна |
| Поле суммы | Обязательное для построения графика | Fallback на количество карточек при отсутствии суммы |
| Определение won/lost | Ручной выбор пользователем | Автоопределение по `column_type = 'done'` + алерт при неоднозначности |

### P-004. Первая доска по умолчанию может быть нерелевантной

| Поле | Значение |
|------|----------|
| **Критичность** | Существенная |
| **Описание** | Система по умолчанию включает первую доску пространства, которая может не быть CRM-доской. |
| **Цитата CPO** | _"По умолчанию включаем первую, которая не нужна нам."_ |
