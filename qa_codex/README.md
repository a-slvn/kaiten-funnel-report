# QA Codex

Артефакты QA-ревью по проекту `funnel_report`.

- `findings-review.md` — findings review по текущей реализации, с приоритетами и ссылками на код.
- `test-cases.md` — набор базовых регрессионных и продуктовых тест-кейсов.
- `exploratory-charter.md` — charter для ручного exploratory-прохода.
- `verification-log.md` — что именно было запущено локально и с каким результатом.

Фокус этого прогона:

- экран отчета `/reports/funnel`
- фильтры
- настройка воронки
- drilldown
- базовые инженерные проверки (`vitest`, `eslint`, `next build`)
