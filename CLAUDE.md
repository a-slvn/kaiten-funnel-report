# Проект: Отчёт «Воронка продаж» для Kaiten CRM

## Контекст
Kaiten — платформа для управления проектами и работой. Используется как для PM, так и для CRM-задач.
Продукт поддерживает доски с настраиваемыми колонками, карточки с кастомными полями, историю перемещений.

## Цель проекта
Разработать отчёт «Воронка продаж», который визуализирует прохождение сделок через этапы воронки и помогает руководителям отделов продаж принимать решения на основе данных.

## Фаза
Исследование и сбор требований.

## Агенты проекта
- **product-manager** — исследование фичи, конкурентный анализ, user stories, PRD
- **business-analyst** — метрики воронки, модель данных, расчёты, спецификация отчёта

## Структура артефактов
Все артефакты складываются в корень проекта:
- `research/` — конкурентный анализ, обзоры
- `requirements/` — PRD, user stories, спецификации
- `data-model/` — модель данных, метрики, расчёты

## Стандарты документов
Ориентир по качеству и глубине спецификации — файл `/Users/slvn/ai/kaiten/feature/suggest-idea-task.md`.
Документы пишутся на русском языке. Формат — Markdown с таблицами.

## Kaiten: техническая база
- Доски с настраиваемыми колонками (этапы воронки)
- Карточки = сделки/задачи
- Перемещение карточки между колонками = переход по этапу
- История перемещений с таймстемпами
- Кастомные поля: сумма сделки, ответственный, источник, и т.д.
- Несколько досок со своей структурой колонок
